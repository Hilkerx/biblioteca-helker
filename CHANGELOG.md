## [1.0.0] - 2025-11-03

Esta é a primeira versão oficial do projeto **Silva'sRead**.

### Added (Adicionado)

* Criação da estrutura inicial do projeto com Flask.
* Implementação dos endpoints básicos da API (CRUD):
    * `GET /livros`: Lista todos os livros.
    * `GET /livros/<id>`: Obtém um livro específico por ID.
    * `POST /livros`: Adiciona um novo livro.
    * `PUT /livros/<id>`: Atualiza um livro existente.
    * `DELETE /livros/<id>`: Remove um livro.
* Criação da documentação inicial (`README.md`).
* Configuração dos arquivos de comunidade (`CONTRIBUTING.md`, `CODE_OF_CONDUCT.md`, `SECURITY.md`).
